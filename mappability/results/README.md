# Mappability results

All outputs from the STAR-based mappability pipeline (v1–v8). 58 files total.

---

## File groups

### 1. Per-transcript uniqueness factors (46 files)

`transcript_uniqueness_factors_*.tsv` — one file per simulation condition.  
19,437 rows (one per MANE Select transcript). Generated by the v1–v8 STAR alignment pipeline.

**Columns:**
| Column | Description |
|---|---|
| `transcript_id` | ENST accession (MANE Select) |
| `gene_symbol` | HGNC symbol |
| `transcript_len` | Transcript length (bp) |
| `read_len` | Simulated read length (bp) |
| `n_positions` | Number of positions where a read was placed (= transcript_len − read_len + 1) |
| `uniqueness_factor` | Fraction of reads that map back uniquely (MAPQ = 255) |

**Naming pattern:** `transcript_uniqueness_factors_{reference}_{read_source}_{L{len}bp}.tsv`

| Reference token | Meaning |
|---|---|
| `genomic` | Full genome (GRCh38.p14, NC_ chr names) — v1, v3, v4, v6 |
| `dedup_cds` | Deduplicated CDS sequences (93,088 entries, NP_/XP_ accessions) — v2, v8_b |
| `MANE_RNA` | MANE Select transcriptomes as reference — v5, v8_c |
| `MANE_exonflank` | Reads spanning exon–intron boundaries — v5 |

| Read source token | Meaning |
|---|---|
| *(no token)* | Exon-flank reads (PE) |
| `SE` | Exon-flank reads, single-end |
| `RNA_SE` | Full MANE RNA reads, single-end |
| `RNA_PE` | Full MANE RNA reads, paired-end |
| `RNA_PE_proper` | Paired-end with `--alignEndsType EndToEnd` (no soft-clipping at read ends) |
| `alignIntronMax1` | SE exon-flank reads with `alignIntronMax=1` (genome reference only) |

### 3. Auxiliary tables (3 files)

**`id_conversion_table.tsv`**  
Maps ENST accessions to ENSG, gene symbol, and NP_ / NM_ accessions. Used by downstream scripts to join transcript IDs across annotation sources.

**`chr_name_map.tsv`**  
Two-column map (no header) between UCSC chromosome names (`chr1`, `chrM`, `chrMT`, …) and NCBI RefSeq accessions (`NC_000001.11`, `NC_012920.1`, …). Used to convert between the `chr`-prefixed naming used by GENCODE/UCSC and the `NC_`-prefixed naming used by the STAR genome index and v7 CDS reference.

**`results_summary.tsv`**  
Aggregate summary of all 46 transcript_uniqueness_factors_*.tsv files. One row per simulation condition (46 rows). Generated by `../summarise_results.py`.

**Columns:**
| Column | Description |
|---|---|
| `script` | Version label (v1_a, v2_b, …) |
| `reference` | Reference genome/transcriptome used |
| `read_source` | What sequences reads were simulated from |
| `mode` | SE or PE |
| `read_len_bp` | Simulated read length |
| `alignIntronMax1` | Whether intron-spanning was suppressed |
| `n_total` | Total transcripts in the union (19,437) |
| `tr_too_short` | Transcripts shorter than the read length (no reads generated) |
| `n_transcripts` | Transcripts with at least one simulated read |
| `mean_f` | Mean uniqueness factor across all transcripts |
| `f_eq_1` | Count with UF = 1.0 |
| `f_ge_0.90` | Count with UF ≥ 0.90 |
| `f_0.10_to_0.90` | Count 0.10 ≤ UF < 0.90 |
| `f_lt_0.10` | Count with UF < 0.10 |
| `f_eq_0` | Count with UF = 0.0 (includes non-coding in CDS-reference conditions) |

**`cds_summary.tsv`** (generated by `../summarise_results_cds.py`)  
One row per v7 condition (10 rows). Covers the 10 `cds_uniqueness_factors_*.tsv` files.

| Column | Description |
|---|---|
| `script` | Version label (v7_a … v7_j) |
| `reference` | `full genome` or `CDS self` |
| `read_source` | Always `CDS reads` |
| `mode` | SE or PE |
| `read_len_bp` | Simulated read length |
| `n_total` | Total CDS entries in the union (92,938) |
| `n_too_short` | CDS entries shorter than the read length |
| `n_cds` | CDS entries with at least one simulated read |
| `mean_uf` | Mean uniqueness factor across all CDS entries |
| `f_eq_1` | Count with UF = 1.0 |
| `f_ge_0.90` | Count with UF ≥ 0.90 |
| `f_0.10_to_0.90` | Count 0.10 ≤ UF < 0.90 |
| `f_lt_0.10` | Count with UF < 0.10 |
| `f_eq_0` | Count with UF = 0.0 |
| `total_positions` | Sum of n_positions across all CDS entries |
| `total_unique` | Total reads mapping back uniquely |
| `total_multi` | Total reads mapping back non-uniquely |
| `total_unmapped` | Total reads that did not map back |
| `global_uf` | total_unique / total_positions (position-weighted UF) |

**Key result:**

| Condition | mean_uf | f=1 | f=0 | global_uf |
|---|---|---|---|---|
| CDS reads → genome (SE/PE, 75–200 bp) | 0.879–0.895 | 60–66K | ~5,900 | 0.891–0.903 |
| CDS reads → CDS self (SE/PE, 100 bp) | 0.109–0.110 | ~5,960 | ~51,700 | 0.056–0.057 |

When CDS reads align to the genome, ~89% map uniquely — the genomic context (introns, flanks, UTRs) provides the sequence space needed to distinguish paralogs. When the same reads align back to the CDS reference itself, only ~5.6% are unique (global_uf): ~56% of CDS entries score UF = 0, meaning their coding sequences are entirely shared with at least one other CDS entry in the deduplicated reference. This is the fundamental coding-sequence redundancy that UTR k-mers rescue in the MANE-reference settings (v5/v8_c).
